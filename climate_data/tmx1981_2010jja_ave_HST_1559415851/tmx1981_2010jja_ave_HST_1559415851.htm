Where these files come from

The files in this bundled were downloaded from the California Climate Commons, and in particular they represent portions of:
  • Dataset 2014 California BCM (Basin Characterization Model) Downscaled Climate and Hydrology - 30-year Summaries
  • Data Variable tmx - Maximum monthly temperature
To see more information about the Dataset or the Data Variable, or to leave comments or discuss the data, click on the links above.

Extracted subset of dataset

The full dataset may contain many related layers for that data variable, depending on factors (parameters) such as which Global Climate Model or Emission Scenario is used, a time period over which values have been averaged, perhaps a month of the year or a species. The parameters which were selected for this download are described above.

The full dataset may also cover a larger geographic area than was needed for download; the downloader may have selected a rectangular portion of the overall dataset to extract and download - the actual downloaded geographic area is described above.

Overview of the files in this package

The download consists of two main files - a data file containing the raw values, and an image or map file with a matching visible representation using colors to represent values.
  • The Data File is in GeoTIFF format, which is a variation of the common TIFF image format with the addition of embedded geographic location information which can be used by some software to display or analyze the data correctly. Each 270m x 270m grid cell is represented by one TIFF pixel.
    Note however that the TIFF data file contains data values in single precision (4 byte) floating point format; while that is a legal option for TIFF files and is accepted by some GIS software, some image display programs may not understand the floating point variation of TIFF and may refuse to display the file as an image, or may display a random noise looking image.
  • There is also a more easily viewable PNG image file, covering the same grid cells but assigning a color value to each cell (each data cell is shown as one pixel). This file uses a particular color scheme (or "symbology" to represent given data values as given colors. Using GIS software on the Data File, you can generate other images with different colors schemes.
  • The download bundle also contains an auxilliary "World File" for each of the above main files, and a "Projection" file which serves for both. These allow some software to make better use of the data and image files.
  • And of course, there is this file, describing the other downloaded files. There is also a PNG file and a GIF file used within this page.
